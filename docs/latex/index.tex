The {\bfseries{MAINBOARD}} subsystem is the central controller for the AVA LLC system. It coordinates communication between vehicle subsystems via CAN and UART, managing the following components\+:
\begin{DoxyItemize}
\item THROTTLE\+\_\+\+SYSTEM (CAN2\+ANALOG)
\item BOOSTER\+\_\+\+SYSTEM (Brake)
\item STEERING\+\_\+\+SYSTEM (EPS)
\item ULTRASONIC\+\_\+\+SYSTEM (Distance sensing)
\end{DoxyItemize}\hypertarget{index_overview}{}\doxysection{\texorpdfstring{System Overview}{System Overview}}\label{index_overview}
The MAINBOARD interfaces with distributed STM32-\/based modules over the CAN bus and streams telemetry data over UART to a host.


\begin{DoxyItemize}
\item {\bfseries{CAN\+:}} Real-\/time command and feedback network (500 kbps)
\item {\bfseries{UART\+:}} Telemetry and diagnostics (115200 bps)
\item {\bfseries{PWM\+:}} Analog control output for throttle
\item {\bfseries{GPIO\+:}} Safety inputs and system indicators
